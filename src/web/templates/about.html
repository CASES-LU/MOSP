MONARC Objects Sharing Platform

A platform to create, edit and share JSON objects.

The goal of this platform is to gather security related JSON schemas and objects.

You can use any available schemas in order to create shareable JSON objects. It also possible to keep an object private even if our goal is to promote the sharing of information. JSON schemas are always public.

You are welcome to copy, modify or redistribute the source code according to the Affero GPL license.

Objects distributed through this platform are distributed under the open-source license choosed by their editor.
